# Hafez Oracle — Privacy Policy

This repository contains the public Privacy Policy for the Hafez Oracle Android app, intended to be linked from the Google Play Store.

Why this repo
-------------
Your main app repository is private. This small public repo provides a simple, hosted privacy policy page (via GitHub Pages) so you can supply a public privacy policy URL to the Play Store.

How to use
----------
1. After I add the files, enable GitHub Pages for this repository.
2. In the repo Settings → Pages, enable GitHub Pages from the `main` branch (root) or whichever branch you prefer.
3. After Pages builds, your privacy policy will be available at:
   `https://ssldaemon.github.io/hafez-oracle-privacy/` (or `https://<your-username>.github.io/hafez-oracle-privacy/`).
4. Copy that URL into the Google Play Console: App → Policy → App content → Privacy Policy.

Edit contact and details
------------------------
The contact email in the files is set to `shadowsockslady@gmail.com`. If you want a different contact email, edit the files and push the change.
